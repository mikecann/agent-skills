# Task Board Seed

This is a small clean seed project for project-scale deslop evals.

It is intentionally simple but realistic enough to be evolved through several
feature requests by fast agents.

## Starting Shape

- a small task board model
- one hook that owns local state and derived values
- two React components
- a small pure store module with task helpers

## Eval Goal

Repeatedly change requirements until the project turns into a believable ball of
slop, then run `deslop` against it and measure how much clarity it restores.
